# Claude Code Sound Packs

Sound packs for [Claude Code](https://claude.ai/code). Each pack plays audio cues for key events like session start, notifications, prompt submit, and stop.

## Available Packs

- `mario` — Super Mario Bros sounds
- `sc-terran` — StarCraft Terran unit responses
- `spongebob` — SpongeBob SquarePants sounds
- `wow-peasant` — World of Warcraft Peasant unit
- `wow-peon` — World of Warcraft Peon unit
- `xbox` — Xbox startup/achievement sounds

## Usage

With the `sound-packs` skill installed, just ask Claude:

> "List available sound packs"

> "Switch to the lightsaber sound pack"
